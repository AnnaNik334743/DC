Дано 2 файла отчета об успеваемости за разные даты. Описание:
	-- Student ID -- Id пользователя на платформе
	-- Username -- имя пользователя на платформе
	-- Grade -- итоговый прогресс (от 0 до 1)
	-- Grade percent -- итоговый прогресс в процентах
	-- Упражнение n -- текущее количество баллов за упражнение n
	-- Упражнение (Avg) -- среднее количество баллов за упражнения
Остальные поля являются техническими и ценности не представляют

На вход может подаваться неограниченное число "срезов" оценок за различные периоды (для большей наглядности демонстрации рекомендуется сгенерировать некоторое количество дополнительных "срезов". Результаты с течением времени ухудшаться не могут).  

Задача -- построить интерактивную визуализацию (dashboard) средствами Python с помощью которой можно было бы визуализировать:
	-- Общий прогресс пользователей за выбранный промежуток времени
	-- Индивидуальный прогресс пользователя за выбранный промежуток времени
	-- Прогресс пользователей по конкретному заданию за выбранный промежуток времени
	-- Прогресс индивидуального пользователя по конкретному заданию за выбранный промежуток времени
	-- Оценку успешности выполнения конкретного задания (сколько пользователей справилось, сколько не справилось)
	-- Оценку успешности выполнения всех заданий (рейтинг популярности выполнения пользователями каждого задания)
	-- Рейтинг пользователей по прогрессу за определенный промежуток времени (кто больше всех продвинулся за выбранный промежуток времени)

Приветствуются предложения по дополнительным видам визуализации, например, поиск подозрительного прогресса (от 0 до 100 за одну ночь) и тп.
